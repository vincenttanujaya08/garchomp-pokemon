# WebGL Garchomp

This project showcases Gabite, Garchomp, and Mega Garchomp, all rendered using pure WebGL with fully built environments. Our team of three members each modeled one Pok√©mon using quadratic objects and configured the rest of the scene with the glMatrix library.
